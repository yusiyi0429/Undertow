# Undertow · 暗流涌动

一个面向金属乐爱好者的社区 Web 应用。

- **中文名**：暗流涌动
- **英文名**：Undertow

## 功能

- **演出日历** (`/`)：卡片式展示近期国内金属演出
- **乐队百科** (`/bands`)：按流派筛选金属乐队
- **社区动态** (`/community`)：匿名发帖，本地持久化

## 技术栈

- Next.js 15 (App Router)
- React 19 + TypeScript
- Tailwind CSS 4
- shadcn/ui
- Vitest + React Testing Library

## 开发

```bash
npm install
npm run dev
```

## 脚本

```bash
npm run dev      # 启动开发服务器
npm run build    # 生产构建
npm run test     # 运行测试
npm run lint     # 代码检查
```
